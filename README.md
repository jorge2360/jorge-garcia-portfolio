# Jorge García Portfolio

Portafolio profesional orientado a oportunidades junior en desarrollo web, análisis de sistemas, bases de datos y tecnología.

---

## Descripción

Aplicación web desarrollada con React y Tailwind CSS para centralizar proyectos, tecnologías, áreas de interés profesional y enlaces de contacto.

El portafolio está enfocado en mostrar habilidades relacionadas con:

- Desarrollo web frontend y backend
- Bases de datos
- Análisis de sistemas
- Ciberseguridad junior
- Infraestructura y soporte técnico

---

## Tecnologías utilizadas

- React
- Vite
- Tailwind CSS
- JavaScript
- Lucide React
- Git / GitHub

## Proyectos destacados

### React Analytics Dashboard
Dashboard administrativo desarrollado con React, Tailwind CSS y Recharts para visualización de métricas y estadísticas.

### Laravel Task Manager
Sistema de gestión de tareas desarrollado con Laravel, autenticación y CRUD completo.

### API REST Inventario
API REST con PHP y MySQL para gestión de productos, categorías y control de stock.

## Ejecutar proyecto

Clonar repositorio:

```bash
git clone https://github.com/jorge2360/jorge-garcia-portfolio.git
```

Ingresar al proyecto:

```bash
cd jorge-garcia-portfolio
```

Instalar dependencias:

```bash
npm install
```

Ejecutar entorno de desarrollo:

```bash
npm run dev
```

Generar build de producción:

```bash
npm run build
```
